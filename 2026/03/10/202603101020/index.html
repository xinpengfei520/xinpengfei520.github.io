从零开始，手把手教你安装配置 OpenClaw AI 助手，支持 Windows/macOS/Linux 全平台，包含常见问题解决方案

什么是 OpenClaw？

OpenClaw（原名 ClawdBot/Moltbot）是一个开源的 AI 智能助手框架，它能让你通过 WhatsApp、Telegram、Discord、飞书等聊天工具直接与 AI 对话，并让 AI 帮你自动执行各种任务。

简单来说，OpenClaw 就像是你的 24 小时在线私人助理：

  • 📱 多平台支持：微信、飞书、Telegram、WhatsApp、Discord 等
  • 🤖 多模型接入：Claude、GPT、DeepSeek、Qwen、智谱 AI 等
  • 🔧 强大的技能系统：PDF 编辑、浏览器控制、日程管理、文档处理等
  • 🔒 数据安全：完全自托管，数据掌握在自己手中

系统要求

在开始安装之前，请确保你的系统满足以下要求：

  • Node.js：v22 或更高版本
  • 操作系统：
    • macOS（推荐）
    • Linux
    • Windows（需要通过 WSL2）
  • 网络：能够访问 npm 仓库

1. 选择 AI 模型供应商

OpenClaw 支持多种 AI 模型，包括：

  • 国外模型：Claude（Anthropic）、GPT（OpenAI）、Gemini（Google）

根据你的需求选择一个供应商，并输入对应的 API Key。

2. 配置聊天渠道

选择你想要使用的聊天平台：

  • Telegram（推荐，配置简单）
  • WhatsApp
  • Discord
  • 飞书（国内用户推荐）
  • 企业微信
  • 钉钉

命令 功能

openclaw tui 打开终端交互界面

openclaw channels list 查看聊天渠道列表
openclaw memory search 搜索助手的记忆
openclaw gateway restart 重启 Gateway 服务
openclaw plugins list 查看已安装的插件

下一步

安装完成后，你可以：

  1. 配置更多 AI 模型：在 ~/.openclaw/agents/<agentId>/agent/auth-profiles.json 中添加
  2. 安装更多技能插件：使用 openclaw plugins install <plugin-url>
  3. 接入更多聊天平台：运行 openclaw channels add
  4. 自定义工作流：通过 Skills 系统创建自动化任务

总结

OpenClaw 是一个功能强大的 AI 助手框架，通过本教程，你应该已经成功完成了安装和基本配置。记住以下关键点：

  • ✅ 确保 Node.js 版本 ≥ v22
  • ✅ 推荐使用一键安装脚本
  • ✅ Windows 用户建议使用 WSL2
  • ✅ 遇到问题先运行 openclaw doctor 诊断
  • ✅ 国内用户可以使用国内 AI 模型（Qwen、智谱等）

如果在安装过程中遇到其他问题，可以查看 OpenClaw 官方文档 或在 GitHub 上提 Issue。
