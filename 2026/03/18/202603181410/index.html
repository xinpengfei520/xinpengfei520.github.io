硬刚五角大楼丢了 2 亿合同，Anthropic 转手给用户撒钱：Claude Code 额度翻倍薅羊毛指南

发表于 2026-03-18 分类于 阅读 阅读次数： 阅读次数： Valine：

最近 AI 圈最炸裂的新闻，不是哪家公司又发了新模型，而是一家 AI 公司跟美国国防部”闹掰了”。

Anthropic，Claude 背后的公司，拒绝了五角大楼的要求——不允许自家 AI 用于自主武器和大规模监控。代价？一份价值 2 亿美元的军方合同没了，还被贴上了”供应链风险”的标签。

然后事情来了个神转折。

合同丢了不到两周，Anthropic 宣布：Claude 全线产品额度翻倍，包括 Claude Code，持续两周，免费白嫖，所有用户都能享受。

这操作，有点意思。

Anthropic vs 五角大楼：到底发生了什么？

事情要从 2025 年说起。Anthropic 和美国国防部签了一份价值 2 亿美元的合同，Claude 成为唯一一个被批准在军方机密网络中使用的 AI 系统。用途包括情报分析、网络安全、后勤建模。

但 Anthropic 在合同里画了两条红线：

  1. 不能用 Claude 驱动全自主武器——就是说 AI 不能自己做”开枪”的决定，必须有人类参与
  2. 不能用 Claude 做大规模国内监控

五角大楼不买账。国防部长 Pete Hegseth 要求 Anthropic 去掉这些限制，给军方”不受约束的访问权限”。

Anthropic CEO Dario Amodei 的回应很硬：

“我们无法凭良心答应他们的要求。”

于是五角大楼设了最后期限。Anthropic 没退让。2026 年 2 月 27 日，特朗普政府下令联邦机构停止使用 Anthropic 的产品。Hegseth 随即将 Anthropic 列为“供应链风险”——这个标签以前只用在跟外国对手有关联的公司身上。用在一家美国公司身上，史无前例。

3 月 9 日，Anthropic 提起诉讼，称五角大楼的做法侵犯了公司的言论自由和正当程序权利。

37 名来自 OpenAI 和 Google DeepMind 的员工——包括 Google 首席科学家 Jeff Dean——签署了支持 Anthropic 的法庭之友陈述书。连竞争对手的人都站出来了。

代价有多大？

Reuters 报道，供应链风险标签可能让 Anthropic 2026 年的收入减少数十亿美元。一个合作伙伴已经从 Claude 切换到竞品，直接砍掉了预期超过 1 亿美元的收入。还有价值约 1.8 亿美元的金融机构谈判也受到影响。

但硬币的另一面是：在与五角大楼公开对抗后，Claude 的 App 冲上了 Apple App Store 榜首。Anthropic 的年化收入从 140 亿美元飙升到 190 亿美元。”QuitGPT” 运动让大量用户从 ChatGPT 转向 Claude。

拒绝军方 = 用户暴增。 这剧本谁能想到？

额度翻倍：怎么回事？怎么薅？

就在风波最激烈的时候，2026 年 3 月 13 日，Anthropic 宣布了一个限时福利：

“给所有 Claude 用户的一个小感谢：接下来两周，非高峰时段的使用量翻倍。”

这不是客气话，是真金白银。

基本信息

  • 时间： 2026 年 3 月 13 日 - 3 月 27 日（太平洋时间 23:59 截止）
  • 适用范围： Free、Pro、Max、Team 计划（Enterprise 除外）
  • 覆盖产品： Claude 网页版、桌面版、移动版、Claude Code、Cowork、Claude for Excel、Claude for PowerPoint
  • 操作方式： 不需要任何操作，自动生效

什么时段有效？

翻倍只在非高峰时段生效：

时区 高峰时段（不翻倍） 非高峰时段（翻倍）
美东 ET 8:00 AM - 2:00 PM 其余时间
美西 PT 5:00 AM - 11:00 AM 其余时间
北京时间 晚 8:00 - 凌晨 2:00 其余时间

好消息：对中国用户来说，白天工作时间基本都是非高峰时段。 你正常上班写代码的时候，Claude Code 就是翻倍状态。

周末全天都是非高峰。

翻倍到底多了多少？

这里有个坑：Anthropic 从来不公布具体的额度数字。但社区开发者通过逆向分析推算出了大致数据。翻倍意味着你的 5 小时滑动窗口里，可以发送的消息数量变成原来的 2 倍。

对 Claude Code 用户来说，感知最明显。因为 Claude Code 的 Agent 循环非常吃 token——一次搜索、读文件、执行命令的循环，就能消耗掉普通聊天几十倍的额度。翻倍后，一个大型重构任务从”做到一半被限流”变成”一口气跑完”。

最佳薅羊毛姿势

1. 大任务安排在非高峰时段。 代码库分析、大规模重构、长文档处理——这些吃额度的活，挪到下午和晚上做（北京时间白天正好）。

2. 周末集中攻坚。 周末全天翻倍，没有高峰限制。有大项目要做？周末是最佳时间。

3. 额外用量不计入周限制。 翻倍部分是”额外”给的，不会影响你的周用量上限。放心用。

4. 别忘了 Claude Code。 这次翻倍不只是聊天，Claude Code 也包含在内。如果你一直想用 Opus 4.6 跑复杂任务但怕额度不够，现在就是最好的窗口。

为什么是现在？

你可能会问：Anthropic 为什么突然这么大方？

表面原因是”感谢用户”。但往深了想，几个因素叠加在一起：

第一，用户暴增，需要留住人。 QuitGPT 运动带来了大量新用户。但新用户最常遇到的问题就是”刚开始用就被限流了”。翻倍额度是留住他们的最好方式。

第二，非高峰时段 GPU 闲着也是闲着。 Anthropic 的服务器在美国工作时间满负荷运转，但晚上和周末大量 GPU 空闲。与其让算力浪费，不如给用户用，还能赚个好口碑。

第三，回应五角大楼事件。 丢了军方合同后，Anthropic 需要向市场传递一个信号——“我们站在用户这边”。限时翻倍额度，时机刚好。

去年 12 月，Anthropic 也搞过一次类似的圣诞节促销，但那次只给 Pro 和 Max 用户，持续一周。这次扩大到 Free 用户，持续两周，覆盖更多产品。明显是用户获取策略升级了。

各套餐怎么选？

趁这个机会，梳理一下 Claude 的订阅体系：

计划 月费 额度 适合谁
Free $0 基础额度 尝鲜、轻度使用
Pro $20 标准额度 日常开发者
Max 5x $100 Pro 的 5 倍 重度 Claude Code 用户
Max 20x $200 Pro 的 20 倍 全天候 AI 编程

如果你还在 Free 或 Pro，这两周是体验”不被限流”的最佳窗口。感受一下不被打断的 Claude Code 是什么样子。如果觉得好，再决定要不要升级 Max。

一个社区里流传的省钱技巧：同时订阅 2-3 个 AI 服务（比如 Claude Pro + ChatGPT Plus），每个 $20/月，总共 $40-60。这样当一个平台限流时可以切到另一个，比单独订 $100 的 Max 更灵活。当然，如果你重度依赖 Claude Code，Max 5x 的 $100 可能是更好的选择。

写在最后

Anthropic 这波操作，不管是主动还是被动，客观上做了两件事：

第一，在 AI 安全上画了一条线。 在所有竞对都向军方妥协的时候，Anthropic 选择了说不。OpenAI、Google、xAI 都接受了”合法使用”的宽泛条款。只有 Anthropic 坚持要把”自主武器”和”大规模监控”写进禁止条款。你可以说这是理想主义，也可以说这是商业策略，但它确实在用户心中建立了差异化。

第二，把省下来的（或者说赚到的）好感，变成了实实在在的用户福利。 额度翻倍虽然只有两周，但对正在做项目的开发者来说，这两周可能就是关键期。

促销到 3 月 27 日 23:59 PT 结束。还有几天窗口。如果你有积压的大任务、想尝试的新项目、或者一直想跑的代码重构——现在就是最好的时机。
