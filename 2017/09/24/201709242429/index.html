      иҜҙжҳҺпјҡжң¬ж–Үд»Ҙе®һзҺ°з®ҖдҪ“дёӯж–ҮгҖҒз№ҒдҪ“дёӯж–Үе’ҢиӢұиҜӯдёәдҫӢиҝӣиЎҢз®ҖиҰҒи®Іи§ЈгҖӮ

1.е®һзҺ°жҖқи·Ҝ

      жҲ‘д»¬еҸҜд»Ҙйў„е…ҲдҪҝз”ЁSharedPreference жқҘдҝқеӯҳдёҖдёӘиҜӯиЁҖзұ»еһӢзҡ„еҖјпјҢеҪ“з”ЁжҲ·з¬¬дёҖж¬Ўиҝӣе…Ҙ APP ж—¶пјҢжҲ‘д»¬йҖҡиҝҮ Key еҸ–еҮәиҝҷдёӘеҖјпјҢз¬¬дёҖж¬ЎиӮҜе®ҡжҳҜеҸ–дёҚеҲ°зҡ„пјҢиҝҷж—¶жҲ‘д»¬е°Ҷ App зҡ„иҜӯиЁҖи®ҫзҪ®дёәеҪ“еүҚзі»з»ҹй»ҳи®ӨеҖјеҚіеҸҜпјӣеҪ“з”ЁжҲ·йҖҡи®ҫзҪ®еӨҡиҜӯиЁҖж—¶пјҢжҲ‘д»¬е°ҶиҝҷдёӘиҜӯиЁҖеҜ№еә”зҡ„еҖјйҖҡиҝҮSharedPreference дҝқеӯҳеҲ°жң¬ең°еҚіеҸҜпјҢз„¶еҗҺжӯӨж—¶йҮҚеҗҜ APPпјҢи®ҫзҪ®дёәз”ЁжҲ·и®ҫзҪ®зҡ„иҜӯиЁҖеҚіеҸҜпјҢд»ҘеҗҺжҜҸж¬Ўиҝӣе…Ҙ App ж—¶еҸӘйңҖеҸ–еҮәдҝқеӯҳз”ЁжҲ·и®ҫзҪ®зҡ„иҜӯиЁҖи®ҫзҪ®еҚіеҸҜпјҢдёҖиҲ¬и®ҫзҪ®иҜӯиЁҖеҶҷеңЁзЁӢеәҸзҡ„е…ҘеҸЈ Application зҡ„ onCreate() ж–№жі•йҮҢгҖӮ

2.иҮӘе®ҡд№үSharedPreferenceе·Ҙе…·зұ»

      дҪңз”Ёпјҡз”ЁдәҺдҝқеӯҳеҪ“еүҚи®ҫзҪ®иҜӯиЁҖзҡ„зұ»еһӢпјҢжӯӨж–Үд»Ҙ SharedPreference дҝқеӯҳдёәдҫӢпјҢеҪ“з„¶дҪҝз”Ёж•°жҚ®еә“жҲ–иҖ…е…¶д»–дҝқеӯҳж–№ејҸд№ҹжҳҜеҸҜд»Ҙзҡ„гҖӮ

3.иҮӘе®ҡд№үLocaleUtilе·Ҙе…·зұ»

      дҪңз”Ёпјҡз”ЁдәҺи®ҫзҪ®дҝқеӯҳиҜӯиЁҖеҸҠиҺ·еҸ–еҪ“еүҚиҜӯиЁҖпјҢйҮҚеҗҜ APP зӯүж“ҚдҪң

3.1Application дёӯи°ғз”ЁпјҢз”ЁдәҺеҲқе§ӢеҢ–иҜӯиЁҖи®ҫзҪ®

3.2еңЁApplication дёӯйҮҚеҶҷеҰӮдёӢж–№жі•пјҡз”ЁдәҺеҪ“зі»з»ҹи®ҫзҪ®иҜӯиЁҖеҸҳеҢ–ж—¶иҝӣиЎҢиҜӯиЁҖи®ҫзҪ®

3.3еңЁеӨҡиҜӯиЁҖи®ҫзҪ®з•Ңйқўи®ҫзҪ®APPиҜӯиЁҖ

      еҪ“з”ЁжҲ·зӮ№еҮ»дҝқеӯҳж—¶дј е…ҘеҪ“еүҚиҜӯиЁҖеҜ№еә”зҡ„ Index еҖјеҚіеҸҜпјҢжҲ‘иҝҷйҮҢдҪҝз”Ёзҡ„жҳҜдҝқеӯҳдёҖдёӘ Int зұ»еһӢзҡ„еҖјпјҢжҜҸдёҖдёӘ Int зұ»еһӢзҡ„еҖјеҜ№еә”дёҖз§ҚиҜӯиЁҖзҡ„зұ»еһӢпјҢеҪ“з„¶дҪ еҸҜд»Ҙж №жҚ®иҮӘе·ұзҡ„е®һзҺ°дҝқеӯҳдёә String зұ»еһӢд№ҹжҳҜеҸҜд»Ҙзҡ„гҖӮ
