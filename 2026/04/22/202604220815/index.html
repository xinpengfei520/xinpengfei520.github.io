SpaceX 拿下 Cursor 600 亿收购权那天早上，我正用它改一段 React

发表于 2026-04-22 更新于 2026-04-24 分类于 产品 阅读次数： 本文字数： 3.9k 阅读时长 ≈ 4 分钟

大家好，我是飞飞。

今早我是被手机推送叫醒的。SpaceX 宣布拿下了以 600 亿美金整体收购 Cursor 的权利，如果不行权，也要支付 100 亿美金深化合作。

我从床上爬起来，打开笔记本。桌面上 Cursor 的窗口还停在前一晚的位置：一段 React 组件改了一半，右侧 Composer 还挂着 Claude Sonnet 的上下文。

这是我过去三个月雷打不动的工作流。Cursor 的 Composer 改前端，右边另开一个 Claude Code 终端跑后端和脚本。每月两边订阅费加起来 40 美金出头。

我盯着那个 600 亿的数字看了一会儿。比奥特曼 2024 年第一次出价高了三倍，比 Cursor 最近跟 a16z 谈的那轮估值还多 100 亿。然后我做了件自己都觉得有点好笑的事：把 Cursor 的订阅账单翻出来，看了一眼续订日期。

先把这个 600 亿 + 100 亿的结构讲清楚

公告本身有点绕，我在 X 上看了三遍才看懂。

SpaceX 并没有”买下”Cursor，它拿到的是一个”今年晚些时候”以 600 亿美金整体收购 Cursor 的权利（option），也可以选择不行权，但要支付 100 亿美金作为双方合作的补偿。

一句话：600 亿行权，或者 100 亿走人。

这个结构极少见。传统 M&A 里的分手费一般是交易额的 1%-3%，这次硬生生拉到了 17%，而且还被明确包装成”合作费用”而不是”违约金”。

法律上它不是收购，不触发反垄断审查。美国 FTC 盯着大科技公司对 AI 创业公司的投资已经盯了一年多，这个壳子刚好绕开。

商业上它又让 Cursor 的独立路径变得极其昂贵。Cursor 正在跟 a16z、英伟达谈一轮 20 亿美金的融资，估值 500 亿。SpaceX 突然甩一个 600 亿的期权过来，等于给这轮融资定了一个”天花板保险”，同时也让任何其他潜在买家知难而退。

极客公园那篇分析写得挺准：”马斯克用算力换来了一张对 Cursor 的优先认购权”。代价是 SpaceX 把 Colossus 这台 100 万块 H100 算力的集群给 Cursor 用。

我每天开的那个 150 人公司，估值比字节大

Cursor 的母公司叫 Anysphere，2022 年成立，现在只有大概 150 个员工。

让这个数字落地一下：150 个员工，目标估值 500 亿美金。人均估值 3.33 亿美金。

对比一下：英伟达大约 30000 员工，市值 3 万亿，人均 1 亿美金；字节 14 万员工，估值 3000 亿，人均 210 万美金。Cursor 的人均估值，是字节的 150 倍。

为什么资本愿意给这个数字？他们的 ARR 大约每 2 个月翻一倍，今年 Q1 已经做到 20 亿美金，内部预测年底到 60 亿。

我身边做 Web 开发的朋友，过去半年里几乎没有一个还在用 VS Code 裸开。要么 Cursor，要么 Claude Code 的 VS Code 插件，剩下几个在用 Windsurf。Cursor 抓住的是”每天写代码的人”这个人群的注意力，而这波人的付费意愿在 SaaS 行业里是最高的那一档。

Claude Code 正在绕开 Cursor 这一整层

这是我最近几个月真实的使用感受。

Cursor 本质是一个 VS Code 的 fork。它把 Claude 和 GPT 的模型接进 IDE，在编辑器层做了 Composer、Agent、Tab 补全这些产品化动作。它的价值在”IDE + 模型中间这一层”。

问题是 Anthropic 去年底推了 Claude Code。一个命令行里直接跑 Claude 的工具，可以 git worktree、可以 TDD、可以写 PR，根本不在 IDE 里，也就不需要 Cursor 这一层。

我自己的分工最近半年就在悄悄变。Cursor 还开着，但只用它改前端 UI 细节（可视化方便）。后端重构、脚本编写、CI 调试这些，全切到 Claude Code 的终端里，因为它一条命令就能跑完整条流水线。

Anthropic 现在既是 Cursor 的模型供应商（Cursor 的底座是 Claude），又是 Cursor 的直接竞争对手（Claude Code 抢开发者）。Cursor 自己也做了 Composer 这个内生模型试图解耦，但训练能力和 Anthropic 差着一个数量级。这就是为什么它需要 SpaceX 的算力。

所以 SpaceX 这单背后的实话其实挺扎心：Cursor 承认了单靠第三方模型自己活不下去，必须开始自研。

xAI 为什么非要买编程工具：Grok 真的不行

xAI 那边的故事也挺有意思。

马斯克 2023 年 7 月做 xAI 的时候，从 DeepMind、OpenAI、Google Brain 挖了 11 位联合创始人。两年下来，这 11 个人据 36kr 的梳理现在几乎全部离职，最后一个在今年 3 月走。

Grok Code Fast 1 是 xAI 今年 Q1 推出的编程模型，跑 SWE-bench 只有 42.3%。同期 Claude Sonnet 4.6 是 77%，GPT-5 Codex 是 74%，Grok 直接掉出了第一梯队。

我自己在 Grok Code Fast 1 刚发布时也跑过它。让它帮我改一段 Next.js 的路由配置，它给出的答案是 Next.js 12 的写法，而我的项目是 Next.js 15。同样的提示词丢给 Claude Sonnet 4.6，它先反问我项目的 Next 版本，然后才动手。

Musk 3 月份自己在 X 上承认过：”xAI 第一次没有建对，必须从地基开始重建。”

然后 3 月同时挖走了 Cursor 的两个产品负责人 Andrew Milich 和 Jason Ginsburg，两人现在直接向 Musk 和 xAI 总裁 Michael Nicolls 汇报。再一个月，就是现在这个 600 亿的公告。挖人、造势、砸钱买，三步间隔不超过 60 天。

奥特曼两次没买下的公司，Musk 一个期权就锁了

这件事还有一个我觉得挺有戏剧性的侧面。

2024 年和 2025 年，奥特曼两次试图收购 Cursor。第一次出价 150 亿被拒，第二次出价 200 亿还是被拒。OpenAI 的逻辑是”我们自己有 Codex，买 Cursor 是补个前端壳”。

Cursor 创始人 Michael Truell 两次都选了独立。虎嗅那篇稿子的标题起得很准：”奥特曼的’白月光’被截胡了”。

Musk 这次没犯奥特曼的错误。他没让 Truell 面对”卖身 vs 独立”的选择，而是给了一个”合作 + 可选收购”的第三条路：你不想卖，我就给你 100 亿合作费和 100 万块 H100 的算力；你愿意卖，我 600 亿兜底。

一个愿意做整体加注的战略投资者，比一个上来就要并表的买家更好接。

这步棋 Musk 想了很久。3 月挖人、4 月谈结构、21 号发公告，刚好卡在 SpaceX 6 月 IPO 的路演启动之前。

这对每天开 Cursor 的人意味着什么

今早看完评论区那堆讨论，我又花了二十来分钟才把这个问题想清楚。

短期看，Cursor 不会有任何变化。协议条款写的是”今年晚些时候”行权，最快也得到 Q4。如果 Musk 选择 100 亿合作费不行权，Cursor 依然独立运营，但会拿到 Colossus 的算力训自己的 Composer 模型。

中期看，Cursor 有 10% 到 20% 的概率会在底层模型上开始用 Grok。因为 xAI 的模型一旦对 Cursor 免费或低价供给，Cursor 就有动力把一部分调用从 Claude/GPT 切过去降本。

对我这种每天开两个 IDE 的用户，最现实的担心是：如果 Cursor 底层模型从 Sonnet 4.6 换成 Grok 5，质量会不会回退。

我给自己设了一个两个月后的检查点。6 月中 Cursor 如果开始推”试用 Grok 底座”的功能，我会跑 10 个我过去三个月最常踩坑的 prompt，对比 Claude 的输出。如果 Grok 的准确率掉 15% 以上，我就用回纯 Claude Code。不管 Cursor 的产品体验多好，算是给自己留一条止损线。

长期看，这笔交易其实是在告诉我们：AI 编程这个市场只会剩下 3-4 家能自研底层模型的公司。Anthropic、OpenAI、Google，以及现在整合了 Cursor + Colossus 算力的 SpaceX。其他所有在 IDE 层做套壳的工具，都会被迫选边站。

前几天跟一个做基金的朋友聊过这事。他问我，如果你现在手里有 100 万美金，你愿意买 SpaceX 的 IPO 还是买进 Cursor 这轮 500 亿估值的融资？

我想了一会儿说：我会买 SpaceX。Cursor 已经被锁死了，它未来最好的结局是 600 亿被 SpaceX 买走。对于一个 500 亿估值进去的后期投资者，20% 的上行空间不足以覆盖”不行权 Cursor 独立”场景下估值被压回 300 亿的风险。

但这只是我一个个人用户加二手金融观察者的想法。你们怎么看？特别是两类人：一是每天用 Cursor 或 Claude Code 写代码的，你们觉得这笔交易会让工具变好还是变烂？二是看科技股的，你们会把 SpaceX 这个 600 亿 Cursor 期权当成 IPO 的加分项还是减分项？
