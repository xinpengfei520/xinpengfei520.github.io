Anthropic 营收反超 OpenAI：一家"安全公司"怎么闷声干到 300 亿美元的

发表于 2026-04-14 更新于 2026-04-16 分类于 观点 阅读次数： 本文字数： 2.8k 阅读时长 ≈ 3 分钟

大家好，我是飞飞。

上周一个做投资的朋友给我发了张图，是 Anthropic 和 OpenAI 的年化收入对比曲线。两条线在 2026 年 4 月交叉了——粉色的 Anthropic 从下面穿过了青色的 OpenAI。

他说：”你天天用 Claude，你知道这事吗？”

我盯着那张图看了五秒钟。说实话，我知道 Claude 越来越好用了，但我没想到 Anthropic 已经在营收上反超 OpenAI 了。

300 亿美元年化收入。15 个月前这个数字是 10 亿。

这条增长曲线有多离谱

先把时间线拉出来。

2025 年 1 月，Anthropic 的 ARR 大概 10 亿美元。那时候 OpenAI 已经 60 亿了，领先 6 倍。

2025 年底，Anthropic 追到 90 亿。OpenAI 涨到 200 亿。差距从 6 倍缩到 2 倍多，但没人觉得 Anthropic 能翻盘。

2026 年 2 月，140 亿。3 月，190 亿。4 月 7 日，Anthropic 官宣：年化收入突破 300 亿美元。

同一时期，OpenAI 的月营收是 20 亿，年化大概 240-250 亿。

交叉了。

Salesforce 花了 20 年才做到 300 亿年收入。Anthropic 从赚第一块钱到现在，不到 3 年。

Claude Code 是那个不声不响的印钞机

很多人以为 Claude 和 ChatGPT 一样，主要靠聊天赚钱。不是的。

Anthropic 80% 的收入来自企业客户。而企业客户里，最猛的增长引擎是 Claude Code。

Claude Code 是 2025 年 5 月公开发布的。6 个月后 ARR 就过了 10 亿美元。到 2026 年 2 月，这个数字变成了 25 亿。

一个终端里的编程工具，9 个月从零做到 25 亿美元年化收入。

我自己每天写代码用的就是 Claude Code。之前我以为它只是我个人的效率工具。看了这些数字我才意识到，我和全球几百万开发者一起，把 Anthropic 的营收曲线推成了一条垂直线。

现在 GitHub 上 4% 的公开 commit 是 Claude Code 生成的。年底预计会到 20%。

企业级 vs 消费级：两条路的终局

OpenAI 有 9 亿周活用户。这个数字听起来很恐怖。

但 98% 是免费用户。

9 亿人用着 ChatGPT，消耗着巨量的 GPU 算力，但基本不产生收入。OpenAI 今年 2 月甚至试过在 ChatGPT 里插广告，结果被骂惨了。

Anthropic 走的是另一条路。它从一开始就盯着企业客户。2 月的时候，年支出超过 100 万美元的企业客户有 500 家。两个月不到，翻倍到 1000 家。财富 10 强里有 8 家在用 Claude。

企业客户的钱有几个特点：粘性高、续约率高、会扩张。一个企业签了 Claude 的合同，不是用完就走的，是越用越多的。

Ramp 的数据更直接：2026 年 3 月，新购 AI 服务的企业里，65% 选了 Anthropic，只有 32% 选了 OpenAI。

这不是”各有优劣”。这是企业市场的天平已经明确倾斜了。

五角大楼事件：拒绝的人反而赢了

2026 年 2 月底发生了一件事，让 Anthropic 从一家技术公司变成了一个符号。

五角大楼要求 OpenAI 和 Anthropic 都在军方的机密网络上部署 AI 模型。OpenAI 签了。Anthropic 的 CEO Dario Amodei 拒绝了——他说 Claude 不能用于没有人类监督的致命自主作战，也不能用于大规模监控美国人。

五角大楼的回应是把 Anthropic 列为”供应链风险”，禁止所有军方承包商使用 Claude。

然后 QuitGPT 运动爆发了。

ChatGPT 单日卸载量暴涨 295%。超过 70 万用户承诺取消订阅。Claude 冲上了 App Store 免费榜第一。430 多名 Google、OpenAI、Microsoft、Amazon 的员工签了公开信，要求自己的公司也拒绝五角大楼的条款。

Katy Perry 发了条推，晒了她新买的 Claude Pro 订阅，配了个爱心。

我翻了一下 Anthropic 的营收曲线，2 月到 3 月那一段，刚好是从 140 亿跳到 190 亿的时候。一个月涨了 50 亿。

Dario Amodei 大概没想到，拒签一份军方合同，比花几千万打广告管用得多。

烧钱的差距比营收的差距更关键

营收只是表面。真正决定长期格局的是成本结构。

据华尔街日报的预测数据，到 2030 年，OpenAI 每年的模型训练成本预计是 1250 亿美元。Anthropic 的预测是 300 亿美元。

同一场比赛，4 倍的成本差距。

OpenAI 预计 2026 年亏损 140 亿美元，2029 年才能实现正现金流。Anthropic 预计 2027 年就能正现金流——比 OpenAI 早两到三年。

这里面有一个容易被忽略的细节：Anthropic 是唯一一家同时在 AWS Bedrock、Google Cloud Vertex AI 和 Microsoft Azure Foundry 三大云平台上都能用的前沿 AI 模型。OpenAI 只在 Azure 上。这意味着 Anthropic 的分发渠道是 OpenAI 的三倍。

我自己的一个感受

我用 Claude Code 快一年了。这一年里，我的博客写作流水线、代码项目管理、技术调研，全部跑在 Claude 上面。

以前我觉得这只是个人偏好。看完这些数据我意识到，我的偏好不是偶然的。Claude Code 做对了一件事：它不跟你闲聊，它直接帮你写代码、跑测试、提交 commit。你打开终端，说一句话，它就开始干活。

ChatGPT 的 9 亿用户里，绝大部分人每天用它问天气、翻译句子、聊两句哲学。Anthropic 的 1000 家企业客户，每家一年花 100 万美元以上，让 Claude 跑在生产环境里，嵌进自己的工作流。前者是注意力，后者是生产力。最终是后者先把营收做到了 300 亿。

这事还没结束

Anthropic 估值已经到了 3800 亿美元，华尔街预估上市前可能涨到 4000-5000 亿。最快 2026 年 10 月 IPO，募资可能超过 600 亿美元。

同时，它刚跟 Google 和 Broadcom 签了新协议，拿到 3.5 吉瓦的 TPU 算力，2027 年开始上线。这是在为下一代 Claude 模型备粮。

OpenAI 也不是没有反击。它的 Stargate 万亿级基础设施计划和自研 Titan 芯片如果在 2026 年底落地，有可能重新拉开距离。但如果落地不了，巨大的资金压力只会让差距越来越大。

我不知道一年后这两家谁的营收更高。但我知道一件事：一年前所有人都觉得 OpenAI 不可追赶。现在这个叙事已经被推翻了。

推翻它的，是一家从来不怎么上热搜的公司。

你觉得 Anthropic 的企业级路线和 OpenAI 的消费级路线，最终哪条能走得更远？我现在倾向于前者——但我更好奇的是，Claude Code 这种”嵌入工作流”的产品形态，会不会成为 AI 商业化的终极答案。
