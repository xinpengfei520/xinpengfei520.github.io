我天天用 Claude Code 写代码，它母公司 9650 亿美元要上市了

发表于 2026-06-02 更新于 2026-07-16 分类于 观点 阅读次数： 本文字数： 3.1k 阅读时长 ≈ 3 分钟
Anthropic于6月1日向SEC保密提交S-1，估值9650亿美元，年化收入两年从8700万美元涨到470亿美元，增长引擎Claude Code九个月做到25亿美元ARR，文章对比同日官宣科创板的智谱，并分析每月12.5亿美元算力开支与上市后涨价的风险。

哈喽，我是飞飞。

6 月 1 号晚上刷到消息，Anthropic 向 SEC 保密提交了 S-1。就是我每天写代码用的 Claude，它的母公司要上市了。估值 9650 亿美元，差一口气就破万亿。

看到这个数字我愣了几秒。我突然意识到一件事：我每天打开终端敲 claude 那一下，不光是在写代码，还在给这家公司贡献收入。我用它写文章、改代码、搭流水线、跑工作流，这些操作背后的每一次 API 调用，都是 Anthropic 年化收入曲线上的一个小点。

9650 亿美元长什么样

Anthropic 两年前的年化收入是 8700 万美元。2024 年底过了 10 亿。2025 年底到了 90 亿。今年 5 月，这个数字变成了 470 亿。两年翻了 540 倍。

这种增速在软件行业几乎没有先例。Salesforce 花了二十年才做到 300 亿美元年收入，Anthropic 的年化跑速三年不到就追平了。要说清楚一点：年化收入是拿某段时间的收入乘以 12，反映的是当前势头。2025 年 Anthropic 实际落袋的全年收入大约 100 亿美元。但就算打折看，这条增长曲线足够让投行掏钱。

H 轮 650 亿美元，由 Altimeter、Sequoia、Dragoneer 领投，估值超过了 OpenAI 3 月份的 8520 亿。SpaceX 6 月 12 号上市，目标估值 1.75 万亿。OpenAI 瞄着 9 月。Anthropic 大概率秋天。Goldman Sachs 预测今年美股 IPO 总募资额能到 1600 亿美元，去年的 4 倍。三家万亿级公司同年上市，Wedbush 的分析师管这叫「闸门打开」。

增长引擎叫 Claude Code，我天天在帮它贡献收入

这轮增长的核心引擎是 Claude Code。

2025 年 5 月上线，到 11 月年化收入过 10 亿美元。今年 2 月到了 25 亿。9 个月从零做到 25 亿 ARR，Wired 写了一句：widely considered best-in-class。

GitHub 上 4% 的公开 commits 来自 Claude Code，年底预计超过 20%。300,000 家企业客户，Fortune 10 里 8 家在用，Deloitte 给 47 万员工部署了 Claude。80% 的收入来自企业 API。

说白了，这家公司八成的钱是开发者和企业拿 API 调用喂出来的。

我自己就是那些开发者里的一个。过去半年我在 Claude Code 上搭了整套写作流水线，从选题侦察到素材搜集、写稿、润色、配图、分发，七个 skill 串成一条线，写一篇文章从选题到交稿基本不用离开终端。写过扒源码挖隐藏配置的文章，跑过动态工作流让几百个子 agent 并行改代码。CLAUDE.md 改了不下三十版，里面光写作规则就有四万多字，hooks 和 skills 攒了二三十个。

我算过一笔账。Pro 订阅每月 20 美元是固定的。API 调用看用量，写一篇文章跑完全流水线大概消耗几十万到上百万 token，遇到要搜资料、批量抓网页的选题还会更多。一个月下来光我一个人大概给 Anthropic 贡献几百块人民币的收入。乘以 30 万家企业客户，乘以那些年消费超百万美元的大户，数字一下子就撑起来了。

我有一次试着把主力工作流切到另一个国产模型。事情的起因是想省点额度，把日常杂活降到便宜模型上跑。头半天还行，简单任务没什么问题。

但到了下午，一个需要读十几个文件、连续调五六个工具的长任务，模型的上下文到后面完全乱了，工具调用的格式开始飘，有几次返回的 JSON 直接不合法。我手动修了两个小时，第二天还是这样。

两天下来基本什么都没产出，老老实实切回了 Opus。从那以后我再也没动过换模型的念头。

这种绑定让我对 Anthropic 80% 企业收入的粘性有了切身体会。当你的 CLAUDE.md 写了几万字、hooks 定制了十几条、skill 流水线磨了半年、连写文章的语感矩阵都用 Claude 训练过，你的整个工作方式就长在这个生态上面了。要拔出来，代价太大了。

两头话：月烧 12.5 亿的公司撑不撑得住

9650 亿看着猛，但风险也是真实的。

Anthropic 每月付给 SpaceX Colossus 数据中心的算力费是 12.5 亿美元，合同签到 2029 年。Amazon 另外承诺了 250 亿投入。公司预计 2028 年盈亏平衡，比 OpenAI 的 2030 目标早两年，但现在还在亏钱。这个成本结构根本不像软件公司，更像基础设施公司。

治理也复杂。Anthropic 是 PBC（公益公司），头上还有一个信托委员会专门盯着公司「别光顾赚钱」。这让 Anthropic 在负责任 AI 这件事上立得住人设，但华尔街可能嫌治理太复杂，给估值打折。

今年五角大楼直接把 Claude 拉了黑名单。原因很直接：Anthropic 不同意让 AI 在没人盯着的情况下跑武器瞄准和大规模监控。官司还在打。

但话又说回来。被五角大楼拉黑之后，Anthropic 的企业端增长反而加速了。Claude 一度冲到美区 App Store 免费榜第一。500 多家客户年消费超过 100 万美元。

实话讲，站在用户角度，我觉得这个估值最硬的底座是开发者绑定。跑分会被追平，参数量会被刷新，但当几十万开发者的工作流长在你的生态里，那个迁移成本比任何技术护城河都实在。

同一天，智谱也官宣了科创板

6 月 1 号这个日子巧。Anthropic 在美国交 S-1 的同一天，智谱在中国宣布要上科创板。

智谱今年 1 月已经在港股上市，股价最高涨了 16 倍，市值一度摸到 8810 亿港元，超过小米、京东和百度。现在计划 A+H 双平台上市，科创板拟募资 150 亿元，120 亿投基座大模型，20 亿投 MaaS 平台。

两家放一起看，体量差距是明摆的。Anthropic 年化收入 470 亿美元，智谱 2025 全年营收 7.24 亿元人民币，粗算差了 400 倍。但估值差距只有 10 倍左右（智谱港股市值约 910 亿美元）。简单讲就是港股 A 股能买的纯 AI 大模型标的太少，物以稀为贵。

不过智谱有一个信号值得留意：API 涨价 83% 之后，调用量反涨了 400%，量价齐升。GLM Coding Plan 付费开发者到了 24.2 万，ARR 突破 2.5 亿美元。CEO 张鹏明确说要对标 Anthropic 走 B 端 API 路线。MiniMax 也同步启动了科创板上市辅导。

我自己也在日常用智谱的模型处理一些对话和翻译类小任务，国产模型这两年进步确实快。但要说写代码、跑长链路的 agent 任务，目前还是 Claude 稳一档。这个差距正在缩小，不过到产品力真正追平、能让我把工作流搬过去的那天，还有距离。

中美两边的头部 AI 公司，都在押同一个方向：靠开发者和企业 API 吃饭。

上市之后，我盯着两件具体的事

作为每天都在用 Claude Code 的人，IPO 对我意味着两件具体的事。

好的那面：更多资金进基础设施。过去几个月我自己就碰到过好几次高峰期限流，一个长任务跑到一半被卡住，等十几分钟才恢复。Anthropic 自己也承认过需求对基础设施造成了压力。上市融到的钱砸进去，这个体验应该能改善。

让我悬着心的那面：上市公司有利润压力。现在还在亏钱的 Anthropic 迟早要给股东交代。API 会不会涨价，Pro 订阅会不会加档，Claude Code 的免费额度会不会收窄。我不确定，但以我现在的绑定深度，一旦涨价我没地方跑，只能认。

所以我打算盯着两件事：秋天正式定价时 Anthropic 给出什么样的财务展望，以及上市之后头一个季度 Claude Code 的定价有没有变动。

涨了多少、哪个档位先动、企业客户和个人开发者是不是同步调，这些细节决定了我接下来的工作流要不要做备选方案。

你天天用的工具值多少钱，有时候你比投行更清楚。

你现在的主力工作流绑在哪家模型上？如果它明天涨价 50%，你切得动吗？
