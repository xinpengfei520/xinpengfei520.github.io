MCP、Tool Calling、Function Calling 到底什么关系？我用自己接的几个 MCP 把它讲清楚

发表于 2026-06-27 更新于 2026-07-02 分类于 AI 本文字数： 3.4k 阅读时长 ≈ 3 分钟

MCP 不替代 Function Calling，两者在不同层。Function Calling 是模型「说」要调哪个工具、参数是啥，只出一个 JSON 不负责执行；Tool Calling 是它的口语化叫法。MCP 是上面一层标准协议，用 JSON-RPC 把工具的声明和执行都标准化，一个 Claude Code 同时挂 banana-image、sync-assistant 多个 server 连封面生成、多平台分发，就是这套底层能力的活样本。
