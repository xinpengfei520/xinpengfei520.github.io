Claude Code 从 Pro 套餐里消失了，GitHub Copilot 也开始暂停注册了

发表于 2026-04-23 更新于 2026-04-24 分类于 产品 阅读次数： 本文字数： 4.4k 阅读时长 ≈ 4 分钟

大家好，我是飞飞。

我自己现在是 Claude Max 5x 订阅者，$100 一个月那档。所以 4 月 20 日 GitHub 宣布暂停 Copilot 个人订阅、4 月 21 日下午 Anthropic 短暂把 Claude Code 从 Pro $20 套餐页面里下架的时候，我账号本身没受影响。

但这两件事连着发生，比”具体哪个套餐怎么改”更值得盯。

我写这篇的时候是 4 月 23 号下午。过去这一周，两家全球最大的 AI 编程工具供应商，在相隔不到 36 小时内，分别对他们的个人低价套餐动了刀。两家几乎同时动手，不像是各自的商业决策，更像是共同撞到了同一个现实：每月 10 块、20 块让用户随便跑 AI 写代码这种定价模型，已经撑不住了。

先把这周的三件事按顺序拆开

事情不是一夜发生的。往前倒四天。

4 月 13 日，GitHub 先把 Copilot Pro 的免费试用关了，理由写的是”调查滥用行为”。这一步没太多人注意。

4 月 20 日，GitHub VP of Product Joe Binder 发了篇官方博客，宣布三件事：Copilot Pro、Pro+、Student 三档个人套餐的新注册全部暂停；现有用户的使用限额收紧；Opus 系列模型从 Pro 套餐里彻底拿掉，Opus 4.7 只保留在 Pro+。Binder 这段话挺扎心的原文：

“It’s now common for a handful of requests to incur costs that exceed the plan price.”

翻译过来：几个请求烧掉的算力成本，就能超过用户整个月交的订阅费。

4 月 21 日下午，Anthropic 的定价页面悄悄改了。原本写着 “Pro plan includes Claude Code” 的那一行被拿掉，支持文档里 “Using Claude Code with your Pro or Max plan” 改成了只剩 Max。Amol Avasare 在 Twitter 上的解释同样坦诚：

“Engagement per subscriber is way up. We’ve made small adjustments along the way (weekly caps, tighter limits at peak), but usage has changed a lot and our current plans weren’t built for this.”

几个小时之后 Anthropic 又把改动回滚了，说是在做 A/B 实验。但 The Register 留了截图：定价页 Pro 那一列，Claude Code 原本是打勾，改完以后变成了叉。

三件事按时间排下来，中间没有任何停顿。

一个 agent 跑 8 小时，能烧穿整月订阅

要理解这周为什么会这样，得先想清楚一件事：我们现在用 AI 编程的方式，跟两年前设计这些订阅套餐时，已经是两个物种了。

2023 年的 Copilot 做的是”代码补全”。你可以理解你在 VS Code 里敲一行，它给你补半行。每次交互几十到几百个 token，非常便宜。

我之前使用过两年多的 Copilot，使用场景是 Android Studio+ Copilot 插件，代码补全确实很爽，后来也使用过 Gemini 和其它的代码补全插件，总觉得差点意思。

到了 2025 年下半年，Agentic 工作流开始成主流。开发者打开 Claude Code 或者 Copilot CLI，丢过去一句：”帮我重构这个模块，跑测试、修 bug、改到全绿”，然后就走开。Agent 自己 git worktree、自己跑测试、自己 diff 自己的错误、自己再改一轮。一跑两三小时不少见。

我也暂停了 Copilot Pro 的订阅，换成了 Claude Code 订阅。

Where’s Your Ed At 拿到的微软内部泄露文件里有一句话：GitHub Copilot 的周运行成本，从 2026 年 1 月到现在，已经翻了一倍。这还只是四个月的变化。

更尴尬的是 GitHub 自己的 /fleet 命令，那是他们推的并行 agent 功能，现在被写进官方博客的”请节制使用”清单。自家推的功能，反过来求用户少用，能说明的东西不多。

Ars Technica 评论区有个真实用户反馈得更直接：Pro 订阅下跑 Claude Code 大概 30 到 60 分钟就撞周限，然后要等 3 到 4 小时才能重置。他买的年费是当生日礼物送自己的，算下来基本没怎么用过。

$20 一个月让你跑 agent，原本就是在亏本做。过去是流量换增长可以认，今年每家开始算账了，就认不起了。

我以前带朋友入门的那条路，这周断了

这段是我写这篇的时候最有感觉的地方。

过去半年，身边陆续有几个朋友来问：”我想试试 AI 编程，该买什么？”我的标准答复一直是：先上 Claude Pro $20 一个月，搭配 Claude Code，够你学一个月，不喜欢下个月取消就行。

这是一条便宜、低门槛、还能真正体验到 agent 工作流的路径。这条路在这周的操作里突然走不通了。

4 月 21 号下午之后注册 Pro 的新用户，打开 claude.ai/pricing 看到的 Pro 套餐那一栏，Claude Code 的位置上画的是叉。虽然 Anthropic 当晚把这个改动滚了回去，但 The Register 和 Ars Technica 都说这是明显的 A/B 试水，意图很清楚：测市场对把 Claude Code 从 Pro 拿掉的接受度。

往深了看更别扭。Anthropic 2026 年 2 月刚跟 Amazon 签了 250 亿美金的 5GW 算力协议，但协议落地需要时间，机房要建、电要通、芯片要装。眼下的缺口只能靠”限流 + 砍入门套餐”来撑过去。

Reddit 上已经有 Copilot Student 套餐的用户晒出截图，说他们除 Haiku 以外的所有 Claude 模型都被默默拿掉了。Student 套餐本来就是给学生准备的。

我自己现在还在 Max 5x 上，不受影响。但下次再有朋友问”入门买什么”，答案一下子变得不体面了。让他们直接上 Max 5x $100，对一个还没确认自己要不要学 AI 编程的人来说不是小钱。让他们用 Copilot Free，跑不了 Agentic 工作流。说到底就是入门门槛被抬高了一档，原来 $20 能做的事，现在要 $100 才能完整做到。

OpenAI 这一周莫名其妙地捡了个红利

两家巨头同时收紧的时候，OpenAI 的 Codex 负责人 Tibo（Thibault Sottiaux）发了条 tweet，正面接话：

“Codex will continue to be available both in the FREE and PLUS ($20) plans. We have the compute and efficient models to support it. For important changes, we will engage with the community well ahead of making them.”

关键信息两条：ChatGPT Plus $20 可以继续用 Codex 跑 Agentic 工作流；”we have the compute” 这五个字在这个时间点说出来，分量很重。

我自己倒没有立刻切过去。Claude Opus 4.7 的代码质量对我来说还是目前最好的。但如果我是一个今天才想入门的人，在”Claude Pro $20 几乎不能用 Claude Code”和”ChatGPT Plus $20 还能用 Codex”之间选，选后者毫无疑问。

短期看，Codex 会在 4 月到 6 月之间吃一波 Copilot 和 Claude 的流失用户。能不能留住是另一回事，那要看 Anthropic 和 GitHub 把新供给补上来的速度。

token 计费要来了，这件事基本是共识

最让开发者心里发毛的其实是第二份信息：泄露文件里微软计划 6 月把所有 Copilot 用户切到 token-based billing，按实际 token 烧多少收多少，扁平订阅彻底告别。

Anthropic 这边更早。The Information 3 月就披露了 Anthropic 对企业用户切成 per-million-token 计费。个人 Pro 端这次动作，本质上是给全量切过去前的铺垫。

对偶尔用一下 AI 辅助写代码的人，或者学生党里只写作业的那拨，按 token 计费其实是好事。以前 $20 是下限，按量算可能一个月只花 $5，反而省了。

对每天跑 agent 的重度用户，比如我自己，反过来就是硬冲击。Max 20x 现在 $200 封顶，切到 token 计费之后，如果我一个月实际烧了 $500 的 token，就要再补 $300。

Anthropic 最近的 Opus 4.7 定价是输入 $5 / 百万 token，输出 $25 / 百万 token，带 7.5x 请求乘数（促销到 4 月 30 日）。光是这个乘数就够我盯着 /cost 面板看半天。

按 token 计费长期来看对整个行业更健康，定价跟成本终于接上了。但对”每月花 $200 跑 agent 不心疼”这种使用习惯，是硬冲击。

写到这儿还有个我自己没想清楚的问题

Raycast 广告那件事还没过去一个月。3 月底 Copilot 被发现在用户 PR 里插入 Raycast 的推广 “tip”，11000 多个 PR 受影响，GitHub VP Martin Woodward 出来道歉说这个行为”变得有点 icky”。紧接着 4 月就是停注册 + 收紧限额。两件事中间隔不到三周，开发者社区对 Copilot 的信任感已经被连击了两次。

所以我写到这儿最想问你们一个问题。

如果你今天带一个完全没用过 AI 编程的朋友入门，你会让他买什么？原来那条 $20/月 的 Claude Pro + Claude Code 路径现在很难走通。

Copilot Pro 新用户注册不进去。Max 5x 一个月 $100 对入门者偏贵。ChatGPT Plus + Codex 算是当下最平衡的选择，但又把朋友绑进了另一家的生态。

评论区告诉我你们最近的答案。特别想听两类人的：一是每天跑 agent 的重度用户，看这周这些调整你的月度账单上会不会多一行；二是最近刚入门或者正准备入门的新手，你现在会先上哪家。
