结果今天，它就上线了。

内容工厂 0.28.1 更新，补上了一键发布这条。你在编辑器里写完、配完图，点一下，就能把整篇稿子连排版带封面，直接送进你自己公众号后台的草稿箱。

昨天还在打磨的那一步，今天接上了

0.28.1 就是把最后这段接上。从雷达里看到一个选题，到稿子躺进公众号草稿箱，中间不用再离开内容工厂。说白了，以前是写完自己搬，现在是写完它帮你送进门。

我这台 Mac 只剩 83G，自己扫了遍盘才发现：吃硬盘的全是我天天用的 AI 工具

发表于 2026-06-03 更新于 2026-06-10 分类于 工具 本文字数： 3.1k 阅读时长 ≈ 3 分钟

哈喽，我是飞飞。

昨天刷推，看到卡兹克发了条东西，标题就够狠：为了不花那 120 刀，他把电脑清理软件做成了一个开源 skill。点进去看，更上头。他让 Codex 对自己的 MacBook 做了一次只读扫描，扫出一堆能清的东西，激进点能腾出 140G。而他原本想买的那个收费软件 CleanMyMac，扫了半天只认出 15.8G。

同一台机器，一个说能清 140G，一个说只有 15.8G，差了快十倍。我看到这个数字，第一反应是：不至于吧？

然后我想起来，我这台 Mac 也好久没收拾了，最近存储一直在飘红。干脆，我也照他这个思路，自己跑一遍。

卡兹克那个 skill，戳中我的是三色分级

他这个 skill 的路子不复杂。

Codex 先对硬盘做一次只读扫描，把哪些目录占了多少空间摸清楚，然后生成一个能点的 HTML 报告。报告里每一项都标了颜色：绿灯是可以放心清的，黄灯是得你自己拿主意的，红灯是碰都别碰的。最后给你一个按钮，确认了再动手删。

我盯着那个三色分级看了一会儿，越想越觉得这才是重点。我自己写 skill 的时候最头疼的也是这一步，怎么让模型把”拿不准的”老老实实标成拿不准，而不是硬给你一个结论。

清理硬盘这件事，难的从来不是”删”，是”判断哪些能删”。一个 rm 命令谁都会敲，可哪个目录删了明天电脑照常开机、哪个删了某个 app 直接打不开，这中间的判断才值钱。卡兹克这个 skill，等于把这层判断摊开给你看。CleanMyMac 把这层判断锁进了一个黑盒，只敢清它有把握的那一小撮。

哈喽，我是飞飞。

做公众号这一年多，我最怕的不是写，是打开后台那一刻，光标在标题栏闪着，我不知道今天该写什么。

等我刷完五六个平台、好不容易锁定一个选题，热点已经凉了一半。就算选题定了，找资料一个钟头，写稿两三个钟头，写完还要找图、抠图、做封面，再去后台一遍遍调样式。一篇文章，一个下午就没了。

真正让我动手的是今年三月的一个晚上。我刷到一个特别好的选题，想着第二天写，结果第二天一忙就忘了。等我再想起来，这事已经被七八个号写过了，热度全过去了。那天我盯着自己空荡荡的文档坐了快二十分钟，心里特别不是滋味。

这套循环我受够了。所以过去几个月，我把自己天天在用的那套 AI 写作流水线，做成了一个网页工具，叫「内容工厂」。今天开放公测。

内容工厂就是把那套东西搬上了网页。不用装环境，不用懂命令行，打开浏览器就能用。说白了，就是我自己每天在用的工具，顺手做成了你也能用的样子。

9650 亿美元长什么样

Anthropic 两年前的年化收入是 8700 万美元。2024 年底过了 10 亿。2025 年底到了 90 亿。今年 5 月，这个数字变成了 470 亿。两年翻了 540 倍。

这种增速在软件行业几乎没有先例。Salesforce 花了二十年才做到 300 亿美元年收入，Anthropic 的年化跑速三年不到就追平了。要说清楚一点：年化收入是拿某段时间的收入乘以 12，反映的是当前势头。2025 年 Anthropic 实际落袋的全年收入大约 100 亿美元。但就算打折看，这条增长曲线足够让投行掏钱。

H 轮 650 亿美元，由 Altimeter、Sequoia、Dragoneer 领投，估值超过了 OpenAI 3 月份的 8520 亿。SpaceX 6 月 12 号上市，目标估值 1.75 万亿。OpenAI 瞄着 9 月。Anthropic 大概率秋天。Goldman Sachs 预测今年美股 IPO 总募资额能到 1600 亿美元，去年的 4 倍。三家万亿级公司同年上市，Wedbush 的分析师管这叫「闸门打开」。

Copilot 把包月改成按 token 计费惹翻开发者，我这种早跳车的来算笔账

发表于 2026-06-01 更新于 2026-06-10 分类于 观点 本文字数： 3k 阅读时长 ≈ 3 分钟

哈喽，我是飞飞。

这两天我刷 GitHub 社区和 X，老看到同一类截图在转：一个开发者把官方的账单预览工具点开，四月份按老规矩算是 39 美元，换成新规则重新一算，902 美元。差不多翻了二十三倍。评论区第一句火出圈的话是「What a joke」，开个什么玩笑。

事情的起因是 GitHub Copilot 从今天，6 月 1 号开始，把计费方式从「包月固定价」改成了「按 token 用多少算多少」。我装 Copilot 那会儿还是它刚火的时候，从那之后我陆陆续续把各家 AI 编程工具的定价都摸了一遍。所以这事我想跟你好好聊聊。先把账算明白，再说说我自己为什么庆幸早早换了阵营。

先把这笔账算明白，Copilot 到底改了什么

老规矩很简单。Copilot Pro 一个月 10 美元，你这个月写多少代码、跟它聊多少次，都是这个价。心里有底。

新规矩换了个内核。还是 10 美元，但这 10 美元现在变成了 1000 个「AI 信用点」，1 个点等于 1 美分。你每发一次请求，它按你这次烧掉的 token 扣点。

token 你可以理解成模型处理文字的最小计价单位。你发进去的、它吐出来的、缓存住的，全算钱。模型越贵，同样一句话扣得越狠。那种会「深度思考」的推理模型，单价能比普通模型高十几倍到几十倍。

套餐价格表面上一分没涨。Pro 还是 10 美元，Pro+ 还是 39，企业版还是那些数字。变的是这笔钱买到的东西，从「一段时间随便用」变成了「一笔额度，用完即止」。点数烧完，Copilot 直接停下不干了，除非你自己手动去后台开「超额预算」。

有一条值得说清楚：最基础的代码补全和下一处编辑建议，还是免费的，不计点。当年让 Copilot 火起来的就是这个边敲边跳出灰字的功能，这块没动。被计价的是那些会自己跑、自己改、自己折腾半天的 agent 能力。
