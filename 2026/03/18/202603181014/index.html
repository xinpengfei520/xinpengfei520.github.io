全网爆火的 OpenClaw 保姆级教程：从安装到 Skills 全搞定

发表于 2026-03-18 更新于 2026-03-22 分类于 阅读 阅读次数： 本文字数： 5.1k 阅读时长 ≈ 5 分钟

你有没有幻想过，有一个 AI 助手 24 小时待命，帮你回邮件、查资料、管日程、跑脚本，甚至替你操作浏览器下单？

这不是科幻电影。这个东西叫 OpenClaw，一个开源的个人 AI 助手。GitHub 上 31 万 Star，全球开发者都在用。你只需要一台电脑、一个 API Key，就能拥有自己的”贾维斯”。

今天这篇文章，我会从零开始，手把手带你走完整个流程：安装、配置、Tools、Skills，一篇全搞定。

OpenClaw 是什么？

一句话：它是跑在你本地的 AI 智能体，通过聊天软件跟你交互。

你在 Telegram 或者 WhatsApp 上给它发消息，它就帮你干活。不是那种只能聊天的 ChatBot，而是真的能操作你的电脑——读写文件、执行命令、控制浏览器、发消息、定时任务，全都行。

它由奥地利开发者 Peter Steinberger 创建，最早叫 Clawdbot，后来改名 Moltbot，再后来正式更名为 OpenClaw。2026 年初爆火，成了开源 AI 领域增长最快的项目之一。

几个关键数据：

  • 310,000+ GitHub Stars
  • 58,000+ Forks
  • 1,200+ 贡献者
  • 支持 50+ 集成（WhatsApp、Telegram、Discord、Slack、iMessage……）
  • 支持 Anthropic Claude、OpenAI GPT、本地模型（Ollama）

MIT 协议，完全免费。你只需要为 AI 模型的 API 调用付费。

安装：10 分钟搞定

环境要求

  • Node.js 22 或更高版本
  • macOS / Windows（WSL2） / Linux 都支持
  • 一个 AI 模型的 API Key（推荐 Anthropic Claude）

安装步骤

引导向导会带你完成几件事：

  1. 选择 AI 模型提供商（Anthropic / OpenAI / Ollama）
  2. 填入 API Key
  3. 连接聊天平台（Telegram / WhatsApp / Discord 等）
  4. 设置基本偏好

整个过程大概 10 分钟。完成后，你就可以在聊天软件里跟你的 AI 助手对话了。

想让它 24 小时在线？

最佳方案：买一台 Mac Mini，装上 OpenClaw，让它一直跑着。成本大概 599 美元硬件 + 每月 10-50 美元 API 费用。

Tools 和 Skills：搞懂这两个概念就够了

很多人装完 OpenClaw 就懵了——Tools 是什么？Skills 又是什么？两者有什么区别？

我用一个比喻来解释：

Tools 是器官，决定 OpenClaw “能不能”做某件事。
Skills 是教科书，教 OpenClaw “怎么组合 Tools”去完成任务。

举个例子。你装了 obsidian 这个 Skill，OpenClaw 知道了怎么管理笔记。但如果你没开启 write 这个 Tool，它连文件都写不了——有知识，没手脚。

所以记住这个关系：Skills 依赖 Tools，Tools 是真正的开关。

26 个 Tools 全解析

OpenClaw 一共有 26 个 Tools，按层级分为两层。

第一层：核心能力（8 个）

这 8 个是基础，大多数人都会开启：

Tool 功能 建议
read 读取文件 开启
write 写入文件 开启
edit 编辑文件 开启
apply_patch 应用代码补丁 开启
exec 执行 Shell 命令 开启，务必开审批
process 管理后台进程 开启
web_search 关键词搜索 开启
web_fetch 读取网页内容 开启

第二层：高级能力（18 个）

这些按需开启：

浏览器控制： browser 让 OpenClaw 操控 Chrome，点按钮、填表单、截图。最新的 3.13 版本支持直接接入你已登录的 Chrome 会话，不需要额外扩展。我用它来比价、查参数，但涉及支付的操作一定自己来。

记忆系统： memory_search 和 memory_get 让它能记住你的偏好。用一周后，它就知道你用什么技术栈、偏好什么语言，不用每次重复说明。

多会话管理： 5 个 sessions 相关工具，让你同时开多个会话处理不同任务，互不干扰。一个讨论产品方案，一个查旅行攻略。

消息发送： message 工具支持向 Discord、Slack、Telegram、WhatsApp、iMessage 发消息。建议只用来给自己发通知，不要代替你跟别人沟通。

定时任务： cron 是让 OpenClaw 变成真正”助手”的关键。每天早上 6:47 给你推一条 Daily Brief，包含今日待办、未回消息、天气预报——这就是 cron + message 的组合。

语音： tts 把回复转成语音消息，支持 ElevenLabs、OpenAI TTS 和 Edge TTS。

53 个官方 Skills 怎么选？

53 个听起来很多，但跟你相关的可能就十来个。重要提示：官方 Skills 默认全部自动加载。 如果你不想要某些 Skill，需要在配置里用白名单模式控制：

按使用场景分类：

笔记类

  • obsidian：操作本地 Obsidian 笔记库
  • notion：云端笔记，无部署限制
  • apple-notes / bear-notes：仅限 macOS 本地运行

如果你的 OpenClaw 跑在云服务器上，选 notion 最省心。

生产力

  • gog：Google 全家桶（Gmail、Calendar、Tasks、Drive、Docs、Sheets），一个 Skill 搞定
  • himalaya：纯 IMAP/SMTP 收发邮件
  • things-mac / apple-reminders / trello：任务管理

如果你用 Google 生态，gog 一个就够了，包含邮件、日历、任务。

开发者工具

  • github：通过 gh CLI 操作 GitHub
  • tmux：管理多个终端会话
  • session-logs：搜索和分析历史对话日志
  • coding-agent：在后台调用其他 AI 编码助手

通讯类

wacli（WhatsApp）、imsg（iMessage）、bird（X/Twitter）、slack、discord——这些 Skill 给 OpenClaw 对应平台的完整访问权限，包括搜索消息历史、同步对话等。

我的建议：通讯类 Skill 谨慎安装。 跟别人沟通的最后一步，永远留给自己。

没错，Skill 就是 Markdown。 不需要写代码、不需要编译、不需要部署。你写清楚步骤，OpenClaw 就按你说的来。

Skill 的加载优先级：

  1. 工作区 Skills（<workspace>/skills）— 最高优先级
  2. 本地 Skills（~/.openclaw/skills）
  3. 官方 Skills（npm 包自带）— 最低优先级

如果同名 Skill 在多个位置都有，高优先级的会覆盖低优先级。

安全注意事项

OpenClaw 跑在你的本地机器上，拥有你授权的一切权限。几个安全建议：

  1. 永远开启 exec 审批。 别怕麻烦，这是底线。
  2. Skills 用白名单模式。 默认全开等于把家门钥匙交给 AI。
  3. 第三方 Skill 先看代码再装。 ClawHub 有扫描但不能 100% 保证安全。
  4. 通讯类操作不要全自动。 给自己发通知可以，代替你跟别人聊天不行。
  5. 支付操作永远手动。 浏览器自动化到加入购物车就够了，结账的按钮自己点。

费用参考

OpenClaw 本身完全免费。你的花费来自 AI 模型 API：

使用程度 月费用
轻度使用 $10 - $30
日常使用 $30 - $70
重度自动化 $100 - $150+

想省钱？用 Ollama 跑本地模型，完全免费，但效果会打折扣。Brave Search 免费版每月 2,000 次请求，日常搜索够用。

写在最后

OpenClaw 不是一个完美的产品。它的配置有门槛，安全需要你自己把关，API 费用也是一笔持续开支。

但它代表了一个方向：AI 不再只是一个对话框，而是一个真正能替你做事的智能体。 它跑在你自己的机器上，数据归你，能力归你，24 小时不下线。

如果你是开发者，强烈建议试一试。不一定要一步到位，从安装开始，开几个核心 Tool，装一两个 Skill，跑几天感受一下。等你体验到每天早上打开 Telegram 就能看到 AI 为你准备好的 Daily Brief，你就会明白——这不是噱头，这是生产力。
