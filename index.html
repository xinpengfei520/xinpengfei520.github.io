看到这个数字我愣了几秒。我突然意识到一件事：我每天打开终端敲 claude 那一下，不光是在写代码，还在给这家公司贡献收入。我用它写文章、改代码、搭流水线、跑工作流，这些操作背后的每一次 API 调用，都是 Anthropic 年化收入曲线上的一个小点。

9650 亿美元长什么样

Anthropic 两年前的年化收入是 8700 万美元。2024 年底过了 10 亿。2025 年底到了 90 亿。今年 5 月，这个数字变成了 470 亿。两年翻了 540 倍。

这种增速在软件行业几乎没有先例。Salesforce 花了二十年才做到 300 亿美元年收入，Anthropic 的年化跑速三年不到就追平了。要说清楚一点：年化收入是拿某段时间的收入乘以 12，反映的是当前势头。2025 年 Anthropic 实际落袋的全年收入大约 100 亿美元。但就算打折看，这条增长曲线足够让投行掏钱。

H 轮 650 亿美元，由 Altimeter、Sequoia、Dragoneer 领投，估值超过了 OpenAI 3 月份的 8520 亿。SpaceX 6 月 12 号上市，目标估值 1.75 万亿。OpenAI 瞄着 9 月。Anthropic 大概率秋天。Goldman Sachs 预测今年美股 IPO 总募资额能到 1600 亿美元，去年的 4 倍。三家万亿级公司同年上市，Wedbush 的分析师管这叫「闸门打开」。

Copilot 把包月改成按 token 计费惹翻开发者，我这种早跳车的来算笔账

发表于 2026-06-01 更新于 2026-06-02 分类于 观点 本文字数： 3k 阅读时长 ≈ 3 分钟

哈喽，我是飞飞。

这两天我刷 GitHub 社区和 X，老看到同一类截图在转：一个开发者把官方的账单预览工具点开，四月份按老规矩算是 39 美元，换成新规则重新一算，902 美元。差不多翻了二十三倍。评论区第一句火出圈的话是「What a joke」，开个什么玩笑。

事情的起因是 GitHub Copilot 从今天，6 月 1 号开始，把计费方式从「包月固定价」改成了「按 token 用多少算多少」。我装 Copilot 那会儿还是它刚火的时候，从那之后我陆陆续续把各家 AI 编程工具的定价都摸了一遍。所以这事我想跟你好好聊聊。先把账算明白，再说说我自己为什么庆幸早早换了阵营。

先把这笔账算明白，Copilot 到底改了什么

老规矩很简单。Copilot Pro 一个月 10 美元，你这个月写多少代码、跟它聊多少次，都是这个价。心里有底。

新规矩换了个内核。还是 10 美元，但这 10 美元现在变成了 1000 个「AI 信用点」，1 个点等于 1 美分。你每发一次请求，它按你这次烧掉的 token 扣点。

token 你可以理解成模型处理文字的最小计价单位。你发进去的、它吐出来的、缓存住的，全算钱。模型越贵，同样一句话扣得越狠。那种会「深度思考」的推理模型，单价能比普通模型高十几倍到几十倍。

套餐价格表面上一分没涨。Pro 还是 10 美元，Pro+ 还是 39，企业版还是那些数字。变的是这笔钱买到的东西，从「一段时间随便用」变成了「一笔额度，用完即止」。点数烧完，Copilot 直接停下不干了，除非你自己手动去后台开「超额预算」。

有一条值得说清楚：最基础的代码补全和下一处编辑建议，还是免费的，不计点。当年让 Copilot 火起来的就是这个边敲边跳出灰字的功能，这块没动。被计价的是那些会自己跑、自己改、自己折腾半天的 agent 能力。

Claude Code 动态工作流上手：一个会话铺开几百个子 agent，是真能打还是噱头

发表于 2026-05-29 更新于 2026-06-02 分类于 编程 本文字数： 2.6k 阅读时长 ≈ 2 分钟

哈喽，我是飞飞。

昨天那篇 Opus 4.8 的结尾，我留了个钩子，说等摸到动态工作流再单独跟你聊。今天就来兑现。

我开了 Max 的权限点进去，找了个真实任务试手，盯着它自己写脚本、一口气铺开几十个子 agent 分头干活。

这东西到底是真能打还是噱头，我盯了两天，心里大概有数了。

先说它到底在干嘛：Claude 自己写脚本，把活拆给几十个分身

说白了，动态工作流就是把「开很多分身一起干活」这件事自动化了。

你给它一个任务，它先自己规划，写出一段编排脚本。然后在同一个会话里同时拉起几十到几百个子 agent，每个啃一小块。跑完汇总给你之前，它还会自己验一遍。

以前这套编排是我手搓的体力活。一个大任务，我得自己拆成几块，开好几个会话分头让它看不同的目录，最后再一段段把结果拼起来。现在这部分它自己包圆了。

我点完任务退到一边，眼看着它在终端里先吐出一段规划，把大任务切成一格一格，然后唰地一下并行拉起一片 agent，每个头上挂着自己负责的那块。那一刻确实有点科幻感，像看一个工头当场招了一群临时工，分完活就各自埋头干了。
