我把每天用的 Claude Code 写作流水线，做成了一个网页工具

发表于 2026-06-02 更新于 2026-07-24 分类于 产品 阅读次数： 本文字数： 2.4k 阅读时长 ≈ 2 分钟
作者把日常用的Claude Code写作流水线做成网页工具「内容工厂」并开放公测，包含聚合AI热点的内容雷达、可切换GLM-5与DeepSeek V4等多模型的长文扩写、左写Markdown右侧手机实时预览的编辑器和一键配图，注册送1000积分，当前专注公众号单平台。

哈喽，我是飞飞。

做公众号这一年多，我最怕的不是写，是打开后台那一刻，光标在标题栏闪着，我不知道今天该写什么。

等我刷完五六个平台、好不容易锁定一个选题，热点已经凉了一半。就算选题定了，找资料一个钟头，写稿两三个钟头，写完还要找图、抠图、做封面，再去后台一遍遍调样式。一篇文章，一个下午就没了。

真正让我动手的是今年三月的一个晚上。我刷到一个特别好的选题，想着第二天写，结果第二天一忙就忘了。等我再想起来，这事已经被七八个号写过了，热度全过去了。那天我盯着自己空荡荡的文档坐了快二十分钟，心里特别不是滋味。

这套循环我受够了。所以过去几个月，我把自己天天在用的那套 AI 写作流水线，做成了一个网页工具，叫「内容工厂」。今天开放公测。

熟悉我的朋友知道，我之前写过好几篇用 Claude Code 搭写作流水线的文章，从选题、搜资料、写稿、润色到配图，七个环节串成一条线。那套东西好用，但门槛不低，得装环境、懂点命令行，还得自己维护一堆配置文件。身边好几个写公众号的朋友看我演示完都说想要，可一听要折腾命令行就打了退堂鼓。

内容工厂就是把那套东西搬上了网页。不用装环境，不用懂命令行，打开浏览器就能用。说白了，就是我自己每天在用的工具，顺手做成了你也能用的样子。

热点不用自己刷，雷达替我盯着

进去第一个页面就是内容雷达。

它把 AI 圈的热点资讯聚合到一条时间线上，预置了 Claude、OpenAI、Gemini、DeepSeek、Cursor 五个主题，顶部点一下关键词就能切。新用户第一次进来会自动开采，几十秒就能看到当天 AI 圈在聊什么，不用做任何配置。

下面这张就是我今天打开时的样子。Claude 31 条、OpenAI 26 条、Gemini 23 条，每条都标了来源，扫一眼就知道这条值不值得写。

对我来说，这一步省掉的就是每天早上刷五个平台那半小时。我现在的习惯是早上泡杯咖啡，打开雷达扫一眼当天的时间线，看到哪条眼睛一亮，直接拎去创作页。三月那次错过选题的事，自从用上雷达就再没发生过。热点不用我自己刷，雷达替我盯着，这是整条流水线的起点。

描述一句话，AI 把长文扩出来

选题有了，点进创作页，迎面是一句「今天想写点什么呢」。

你把选题描述一句，或者粘一段参考素材进去，AI 就能扩写成一篇完整的公众号长文。推荐选题直接来自你刚在雷达里追的那些关键词，省掉来回切换的功夫。

我自己最看重的是多模型这块。下拉框里 GLM-5、DeepSeek V4、Gemini 3、MiniMax M2 几个模型随便切。国内能直连的 DeepSeek、GLM、MiniMax 都在，不用信用卡，也不用担心连不上。每个模型旁边标着费率倍率，按性价比挑就行。

同一个选题，我会让两三个模型各写一版，挑最顺眼的那篇接着改。我自己的体感是，写偏技术的内容 DeepSeek 和 GLM 的逻辑更稳，写偏故事、偏情绪的段落 Gemini 的语感更松弛。换着用，比死磕一个模型出来的东西灵活得多。不被一家 AI 锁死，写出来的东西才不至于千篇一律。

左边写字，右边就是公众号里的样子

稿子出来了，进编辑器。这是我花心思最多的地方。

左边写 Markdown，右边是一个手机预览，公众号的原生样式实时显示，所见即所得。再也不用在后台调一次样式、刷新一次、跳一次。

下面这张就是我用它写上一篇 Claude Code 文章时的界面。左边是正文，右边手机里已经排好版，连封面图都带上了。

以前我在公众号后台改个字号、调个行距，得保存、刷新、再划到对应段落看效果，一篇长文来回折腾几十次。现在右边那个手机预览实时跟着动，改一个字立刻就能看到最终样子。

配图也在这儿。点一下一键配图，AI 按文章内容生成封面或者流程图，背后用的是 Gemini 的图像模型。我最近几篇文章的封面，基本都是它出的，省掉了我以前满网站找图、再担心版权的麻烦。还有 20 多套主题样式，技术、生活、商业三类都覆盖，深色浅色都能切，长文写久了眼睛不那么累。

写完一键复制，排版样式带着走，粘进公众号后台不塌。这一步省掉的，是我以前在后台跟样式较劲的那一个钟头。

注册送 1000 积分，但我得说几句实话

公测期注册就送 1000 积分，大概够生成 50 篇文章再配几十张图。每天登录再 +100，邀请一个朋友 +100，长期用基本不怎么花钱。

每一笔消费都有明细，每生成一篇、每出一张图扣了多少积分，列得清清楚楚，不会莫名其妙扣。站内支持微信赞助换积分，自愿、只能站内消费、不能提现，这点我先讲清楚，免得有人误会成充值理财。

有句实话我得放前面：内容工厂现在只做了公众号一个平台。

做的过程里我最纠结的就是这件事。一开始我也想贪多，小红书、X、视频脚本、知识星球一稿多发，听起来特别诱人。但真动手就发现，每个平台的调性、字数、排版差得太远，一稿多发要么哪个都不顺手，要么质量全打折。我想了很久，决定先把公众号这一个平台做深做稳，别的等做扎实了再说。

你可能在别处见过「一键发布」「多平台一稿多发」这类说法。那些功能我公测前临时下线了，正在打磨，还没正式上线。我不想让你满怀期待进来，结果发现没有，那种落差比不宣传更伤人。所以现在能用的，就是上面这些，一个都没虚标。

它能帮你提速，定稿还得是你

最后说点心里话。

内容工厂能帮你把选题、写稿、配图、排版这几个环节提速，但它替代不了你的判断和审美。AI 负责出稿，你负责定稿，这个顺序我觉得不能反。我自己用它出的初稿，每次也都要再逐句改一遍，把那些套话删掉、把只有我才有的体感和观点加进去。一篇文章里真正打动人的那句话，机器替不了，还得你自己写。

这是我的第一款产品，还在打磨，肯定有不少别扭的地方，说不定你点开就能踩到一两个 bug。公测期我最缺的就是真实反馈。你用着哪儿不顺、想要什么功能，直接在评论区或者后台告诉我，我看到都会认真记下来，能改的会很快迭代。

打开 aigc.vance.xin，注册就送 1000 积分，今天就能写第一篇。

你写公众号时，选题、写稿、配图、排版这四件事，最拖你时间的是哪一个？评论区告诉我，我看看内容工厂现在能不能帮上忙。
